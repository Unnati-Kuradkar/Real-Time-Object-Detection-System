====================================================
RF-DETR OBJECT DETECTION PROJECT
====================================================

Project Name:
RF-DETR Object Detection System

Description:
A Streamlit-based web application that detects objects
from images and videos using the RF-DETR model.

----------------------------------------------------
FEATURES
----------------------------------------------------

✅ Image Object Detection

✅ Video Object Detection

✅ Bounding Boxes

✅ Object Labels

✅ Confidence Scores

✅ Object Count

✅ Object Summary

✅ Fast Video Processing

✅ Streamlit Web Interface

----------------------------------------------------

----------------------------------------------------

Backend:
- Python

AI Model:
- RF-DETR Nano

Libraries:

----------------------------------------------------
STREAMLIT CONFIG
----------------------------------------------------

Create:

.streamlit/config.toml

Add:

[server]
maxUploadSize = 200

This allows larger video uploads.

----------------------------------------------------
INSTALLATION
----------------------------------------------------

1. Clone Repository

git clone <repository-url>

2. Open Project Folder

cd rf-detr-streamlit

3. Install Dependencies

pip install -r requirements.txt

----------------------------------------------------
RUN PROJECT
----------------------------------------------------

streamlit run app.py

OR

python -m streamlit run app.py

----------------------------------------------------
LOCAL URL
----------------------------------------------------

http://localhost:8501

----------------------------------------------------
IMAGE DETECTION WORKFLOW
----------------------------------------------------

1. Select Image Mode

2. Upload Image

Supported Formats:
- JPG
- JPEG
- PNG

3. Model Detects Objects

4. Output Displays:

- Bounding Boxes
- Labels
- Confidence Scores
- Object Count
- Object Summary

Example:

car 0.95
person 0.89
dog 0.92

----------------------------------------------------
VIDEO DETECTION WORKFLOW
----------------------------------------------------

1. Select Video Mode

2. Upload Video

Supported Formats:

- MP4
- AVI
- MOV

3. Click

▶ Start Detection

4. Model Processes Frames

5. Output Displays

- Detected Frames
- Labels
- Confidence Scores
- Object Count Summary

----------------------------------------------------
CONFIDENCE SCORE
----------------------------------------------------

Example:

car 0.91

Meaning:

0.91 = 91% confidence

Confidence Range:

0.90 - 1.00
Excellent

0.70 - 0.89
Good

0.50 - 0.69
Medium

Below 0.50
Low Confidence

----------------------------------------------------
COMMON ERRORS
----------------------------------------------------

ERROR:
ModuleNotFoundError

Fix:
pip install -r requirements.txt

----------------------------------------------------

ERROR:
NameError: sv not defined

Fix:

import supervision as sv

----------------------------------------------------

ERROR:
IndentationError

Fix:
Check spacing carefully.
Use 4 spaces consistently.

----------------------------------------------------

ERROR:
Video Upload Slow

Fix:
Use smaller videos
(<20 MB recommended)

----------------------------------------------------

ERROR:
No Objects Detected

Fix:
Use clear image/video
with visible objects.
